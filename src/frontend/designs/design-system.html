📖 使用指南

如何在项目中正确使用TradeFlow设计系统。

CSS变量

/* 在你的CSS文件中使用变量 */ .my-component { background-color: var(--primary-blue); color: white; padding: var(--spacing-md); border-radius: var(--radius-lg); transition: all var(--transition-fast); } .my-component:hover { background-color: var(--primary-blue-hover); transform: translateY(-2px); box-shadow: var(--shadow-lg); }

响应式断点

/* 移动端优先的响应式设计 */ @media (max-width: 768px) { /* 平板和手机样式 */ } @media (max-width: 480px) { /* 手机样式 */ }

设计原则

  • ✅ 保持设计的一致性，使用统一的色彩、字体和间距
  • ✅ 优先考虑可访问性，确保足够的对比度和可读性
  • ✅ 响应式设计，适配不同设备和屏幕尺寸
  • ✅ 性能优先，避免过度的动画和效果
  • ✅ 用户体验优先，简化交互流程
